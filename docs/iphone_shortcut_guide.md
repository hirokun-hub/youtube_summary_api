# [[iPhone]]ショートカットからYouTube要約APIを利用する方法

このドキュメントは、作成した `youtube-summary-api` を [[iPhone]] の「ショートカット」アプリから呼び出し、YouTubeアプリの「共有」メニューからワンタップで動画の情報を取得する手順を解説します。

## 1. 準備するもの

- **APIのエンドポイントURL**: `https://youtube-summary-api-4ll0.onrender.com/api/v1/summary`
- **あなたのAPIキー**: [[Render]]で設定した、あなた専用のAPIキー


## 2. ショートカット作成手順（改訂版）

「Not Found」エラーを防ぐため、URLを直接指定する、より確実な方法で作成します。

1.  **ショートカットの基本設定**
    -   「ショートカット」アプリを開き、右上の `+` をタップ。
    -   上部のショートカット名をタップ → `詳細` → **`共有シートに表示`** をオンにする。
    -   `共有シートの入力` を `クリア` し、**`URL`** のみを受け取るように設定する。

2.  **アクション①：APIのURLを「テキスト」で指定**
    -   `+ アクションを追加` をタップし、「**テキスト**」アクションを追加します。
    -   テキストボックスに、準備した **APIのエンドポイントURL** を正確に貼り付けます。
        -   `https://youtube-summary-api-4ll0.onrender.com/api/v1/summary`

3.  **アクション②：「URLの内容を取得」でAPIを呼び出す**
    -   `+ アクションを追加` をタップし、「**URLの内容を取得**」アクションを追加します。
    -   `URL` の部分には、前のステップで作成した `テキスト` 変数を選択します。
    -   右向き矢印 `>` をタップして詳細を開き、以下を設定します。
        -   **方法**: `POST`
        -   **ヘッダ**: 
            -   `X-API-KEY`: **あなたのAPIキー**
            -   `Content-Type`: `application/json`
        -   **要求本文**: `JSON` を選択し、以下のように設定します。
            -   キー: `url`
            -   種類: `テキスト`
            -   値: `ショートカットの入力` 変数を選択。（これがYouTubeの共有URLになります）

4.  **アクション③：「辞書の値を取得」で結果を抽出**
    -   `+ アクションを追加` し、「**辞書の値を取得**」を追加します。
    -   `辞書` には `URLの内容` 変数を選択します。
    -   `キー` に `transcript` と入力します。（`title` や `channel_title` も同様に取得可能です）

5.  **アクション④：「結果を表示」で完了**
    -   `+ アクションを追加` し、「**結果を表示**」を追加します。
    -   `辞書の値` 変数が自動で入ります。お好みで「文字起こし結果：」などのテキストを前に追加してください。

6.  **保存**
    -   右上の `完了` をタップして保存します。


## 3. 使い方

1.  YouTubeアプリで、情報を取得したい動画を開きます。
2.  動画の下にある `共有` ボタンをタップします。
3.  共有先アプリの一覧から、作成したショートカットをタップします。
4.  ショートカットが実行され、しばらくすると文字起こしの結果が表示されます。


## 4. トラブルシューティング

#### Q. ショートカットを実行すると「Not Found」と表示される

-   **A1. エンドポイントURLの確認**: ショートカットの「テキスト」アクションに入力したURLが、`/api/v1/summary` まで正確に含まれているか確認してください。ドメイン名だけでは「Not Found」になります。
-   **A2. サーバーがスリープしている**: Renderの無料プランでは、15分アクセスがないとサーバーがスリープします。最初のアクセスはサーバーを起こすために30秒〜1分ほど時間がかかったり、タイムアウトで失敗することがあります。**少し待ってからもう一度試して**みてください。

#### Q. 「認証できませんでした」や「Unauthorized」と表示される

-   **A. APIキーの確認**: 「URLの内容を取得」アクションのヘッダに設定した `X-API-KEY` の値が、Renderで設定したものと完全に一致しているか確認してください。コピー＆ペーストが確実です。

#### Q. YouTubeの共有メニューにショートカットが出てこない

-   **A. ショートカット設定の確認**: ショートカットの `詳細`設定で、`共有シートに表示` がオンになっているか、入力タイプが `URL` を受け付ける設定になっているか、再度確認してください。
