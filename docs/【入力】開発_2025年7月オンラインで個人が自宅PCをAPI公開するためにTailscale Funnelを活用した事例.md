【入力】（開発）_2025年7月オンラインで個人が自宅PCをAPI公開するためにTailscale Funnelを活用した事例

### 概要
[[【法人】Tailscale]] の「Funnel」は、個人のPCやRaspberry Piをインターネット越しに公開するための無料機能であり、安全かつ簡単に HTTPS 経由でアクセス可能な公開 API を作成できる。

本事例では、[[2025-07-05]] 時点で YouTube 字幕を取得するために FastAPI を自宅に立て、それを iPhone Shortcuts から利用可能にする構成として Funnel が活用された。

---

### なぜ Tailscale Funnel は無料なのか？

#### 1. フリーミアム型のビジネスモデル
- [[【法人】Tailscale]] は「**個人利用は無料、チーム利用で課金**」という戦略。
- Funnel は無料提供することで開発者を引き込み、将来的な有料チームプラン（Starter: \$6/月、Premium: \$18/月）への移行を促進する。

#### 2. 運用コストが極めて低い
- Funnel 経由の通信は **端末自身が TLS を終端**し、通信内容の暗号化・復号・転送は行わない。
- Tailscale 側は **WireGuard の鍵配布、TLS証明書の自動発行、メタデータ配信**のような軽量な処理のみ担当。
- 高負荷リレーは DERP（低帯域UDP）に分離されており、大きな帯域負担が発生しない。

#### 3. 提供側のメリット
- **ネットワーク効果**：URL 発行が簡単 → ユーザー増 → Tailnet が広がる。
- **差別化**：競合である [[【法人】ngrok]] などは類似機能を有料提供。Funnel の無料公開は強力な引き。
- **プライバシー尊重**：トラフィック内容の解読を行わず、**GDPR/企業セキュリティ対応が容易**。

---

### 安全性とセキュリティの設計

#### 1. 技術的安全性
- **WireGuard VPN + Let’s Encrypt TLS**：デバイス同士は完全なエンドツーエンド暗号化。
- **自己終端 TLS**：Funnel URL の TLS は**自宅端末で終端**され、外部で改ざんされる余地がない。
- **認可管理**：Tailnet の ACL で Funnel の使用端末・ポートを細かく制御可能（例：`nodeAttrs=["funnel"]` の指定）

#### 2. 既知の注意点
- **証明書 Transparency (CT) ログに記録される**：`*.ts.net` ドメインは公開ログに載るため、Funnel 利用端末はスキャン対象となる可能性あり。
- **対策**：Basic認証・APIトークン・Cloudflare Access の併用を推奨。

### セットアップの簡単さ（ステップバイステップ）

| ステップ | コマンド | 説明 |
|---|---|---|
| ① | `tailscale up` | Tailnet に自宅PCを登録（GUI or CLI） |
| ② | `tailscale funnel 8000` | ポート8000を HTTPS 公開（自動で DNS `xxxxx.ts.net` が発行され、TLS 証明書も即適用） |
| ③ | アクセス確認 | iPhoneや他端末から `https://xxxxx.ts.net/` へアクセス、API が稼働しているか確認 |

> 上記により、**ルーターのポート開放不要／証明書の設定不要／DDNSも不要** で、完全HTTPS化されたAPIが即時完成。

### Funnel の利用制限（無料プラン）

| 項目 | 制限 |
|------|------|
| Funnel 利用 | 無料プランでも利用可能（特別な申請なし） |
| ユーザー数 | 最大 3 人 |
| 接続デバイス | 最大 100 台 |
| アクセス制御 | Tailnet ACL / タグ付けによる制限は無料で使用可 |
| 商用利用 | 個人用途なら可。企業利用は Starter / Premium プランへの移行が必要。

### 実際のユースケース

**目的**：iPhone の Shortcuts から、YouTube 字幕を取得する FastAPI API を呼び出す

**構成**：
- 自宅PC（または Raspberry Pi）で `uvicorn` + `caption_api.py` を起動
- `tailscale funnel 8000` で HTTPS 化された `https://xxxxx.ts.net/caption` を公開
- Shortcuts 側で `URL → Get Contents of URL → Parse JSON → text` で字幕を取得
- 月額コストはゼロ（PC の電気代のみ）
