---
title: "作業ブランチを切り替えてファイルを統合するフロー"
---
flowchart TD
    subgraph "現在のPCでの操作"
    A["Start: masterブランチにローカルファイルが存在"] --> B{"git stash push -u"};
    B --> C["ローカルのファイルを再度一時退避"];
    C --> D{"git fetch origin"};
    D --> E["リモートの全ブランチ情報を取得"];
    E --> F{"git branch -r"};
    F --> G["リモートブランチの一覧を確認し、<br>作業したいブランチ名を見つける"];
    G --> H{"git checkout <ブランチ名>"};
    H --> I["作業ブランチに切り替え"];
    I --> J{"git stash pop"};
    J --> K["退避したファイルを新しいブランチに復元"];
    K --> L{"git add ."};
    L --> M{"git commit"};
    M --> N{"git push origin <ブランチ名>"};
    N --> O["End: 新しいブランチにファイルを追加してPush完了"];
    end
